The python and bash scripts allow you to submit Naive Bayes training jobs to the Barley queue.
1. Copy over the entire Java project to corn. 
2. Edit arimaa_nb_submit.py to configure your parameters -- namely, the range of example sizes to use and the number of trials per size. 
3. Run python arimaa_nb_submit.py to submit the jobs. 
4. While the jobs are queued, check on their status using qstat. See man qstat for more details. 
5. After the jobs are finished, the output will be written to output files (named arimaa_nb_<num-examples>_<trial number>.o<job-id>). 


=========================================================================
If you need to compile or run the Java program, here's how you do it... 
# Compile from root ArimaaBot dir.
javac -d bin -classpath ./mysql-connector-java-5.1.26-bin.jar:./src:./ src/naive_bayes/NBMain.java
javac -d bin -classpath ./mysql-connector-java-5.1.26-bin.jar:./liblinear-1.94.jar:./src:./ src/svm/SVMMain.java  

--------------------------------------------------

# Run from root dir, assuming you have compiled class files in bin/. Of course, configure the arguments to NBMain according to your needs. 
# Arguments: <min-example-set-size> <max-example-set-size> <increment> 
# e.g. to train on example sets of size 10, 15, and 20, use arguments '10 20 5' 
# e.g. to train on a single example set of 20 games, use arguments '20 20 1'
java -Xmx500000000 -classpath ./mysql-connector-java-5.1.26-bin.jar:./bin:./ naive_bayes.NBMain <three-arguments>
